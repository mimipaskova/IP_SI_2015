### Задача 1: Факториел

```
    5
```

Изход:
```
    120
```

Вход:
```
    6
```

Изход:
```
    5
```

### Задача 3: Reverse

число:

Вход:   
```
    654
```

Изход:
```
    456
```

### Задача 4: GCD
Да се напише функция, която връща най-големия общ делител на све числа

Вход:
```
    8 12
```

Изход:
```
    4
```


### Задача 5: x^n
Да се напише функция, която връща x^n
Ако x>1 -> x^n
    x=0 -> 1 
    x<1 ->  1/x^n

Вход:
```
    3 2
```

Изход:
```
    9
```
